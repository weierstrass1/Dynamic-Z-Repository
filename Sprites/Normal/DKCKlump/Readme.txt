You must set up the Pixi Extended ID of the Klump Grenade in this line:

!KlumpGrenadeID = $00

You can edit the following variables:

!MaxResistTime = $30	
!ResistXSpeed = $20
!RecoilXSpeed = $38
!RecoilYSpeed = $C0

Dead SFX can changed here:

!DeadSFX = $40
!DeadSFXAddress = $1DFC|!addr

Laugh SFX can be changed here:

!HitSFX = $3C
!HitSFXAddress = $1DFC|!addr
